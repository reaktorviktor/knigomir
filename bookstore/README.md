# КнигоМир — Книжный интернет-магазин на Django

Дипломная работа: полноценный интернет-магазин книг на Django.

## Структура проекта

```
bookstore/
├── bookstore/          # Настройки проекта
│   ├── settings.py
│   ├── urls.py
│   └── wsgi.py
├── store/              # Главное приложение магазина
│   ├── models.py       # Модели: книги, авторы, корзина, заказы
│   ├── views.py        # Представления
│   ├── urls.py         # URL-маршруты
│   ├── forms.py        # Формы
│   ├── admin.py        # Админ-панель
│   └── templates/store/ # HTML-шаблоны магазина
├── users/              # Приложение пользователей
│   ├── models.py       # Модель профиля
│   ├── views.py        # Регистрация, вход, профиль
│   ├── urls.py
│   ├── forms.py
│   └── templates/users/ # Шаблоны пользователей
├── templates/          # Базовый шаблон
│   └── base.html
├── media/              # Загружаемые файлы (обложки книг, фото)
├── manage.py
├── requirements.txt
├── populate_db.py      # Скрипт заполнения данными
└── README.md
```

## Функционал

- Каталог книг с фильтрацией по категориям и сортировкой
- Страница детальной информации о книге
- Поиск по названию и автору
- Страница автора с его книгами
- Корзина (добавление, изменение количества, удаление)
- Оформление заказа
- История заказов
- Список избранного
- Регистрация и аутентификация
- Профиль пользователя (просмотр и редактирование)
- Отзывы с рейтингом
- Админ-панель Django
- Страницы «О нас» и «Контакты»

## Установка и запуск

### 1. Убедитесь, что установлен Python 3.10+

```bash
python --version
```

### 2. Создайте и активируйте виртуальное окружение

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/Mac:**
```bash
python -m venv venv
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Примените миграции

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Заполните базу данных тестовыми данными

```bash
python populate_db.py
```

Это создаст:
- Суперпользователя: `admin` / `admin123`
- 8 категорий книг
- 8 авторов
- 12 книг

### 6. Запустите сервер

```bash
python manage.py runserver
```

### 7. Откройте браузер

- Сайт: http://127.0.0.1:8000/
- Админ-панель: http://127.0.0.1:8000/admin/

## Технологии

- **Backend:** Python 3.10+, Django 4.2
- **База данных:** SQLite (встроена в Django)
- **Frontend:** Bootstrap 5, Font Awesome, Google Fonts
- **Изображения:** Pillow

## Модели данных

| Модель     | Описание                          |
|------------|-----------------------------------|
| Category   | Категории книг                    |
| Author     | Авторы книг                       |
| Book       | Книги (основная модель)           |
| Review     | Отзывы пользователей              |
| Cart       | Корзина покупателя                |
| CartItem   | Позиция в корзине                 |
| Order      | Заказ                             |
| OrderItem  | Позиция в заказе                  |
| Wishlist   | Список желаемого                  |
| Profile    | Профиль пользователя              |

## Добавление книг через админ-панель

1. Зайдите на http://127.0.0.1:8000/admin/
2. Войдите: `admin` / `admin123`
3. Перейдите в раздел «Книги» → «Добавить»
4. Заполните поля и сохраните

**Важно:** Для поля slug используйте только латинские буквы, цифры и дефисы.
